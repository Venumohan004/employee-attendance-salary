# Employee Attendance & Salary Calculator (MS Excel)

## Description
This project automates employee attendance tracking and salary calculation using Microsoft Excel. It uses Excel formulas such as COUNTIF and VLOOKUP to calculate present days, deductions, and net salary accurately.

## Features
- Employee details management
- Attendance tracking (P/A/L)
- Automatic salary calculation
- Salary report with chart

## Tools Used
- Microsoft Excel
- Microsoft Word
